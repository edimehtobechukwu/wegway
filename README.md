# 🌍 Weg Way

Welcome to **Weg Way** – a beautifully designed, modern web application built for dynamic user experiences.

## ✨ Features

- **Modern Stack**: Powered by React 18, Vite, and JSX.
- **Beautiful UI Components**: Built on top of Radix UI primitives for customizable, accessible, unstyled components.
- **Fluid Animations**: Leveraging `framer-motion` for smooth, delightful interactions.
- **Internationalization**: Full i18n support enabled out of the box to reach a global audience.
- **Responsive Layouts**: Designed to look stunning on any device, from mobile to desktop.

## 🚀 Getting Started

Follow these steps to get the project up and running locally.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

Clone the repository and install the required dependencies:

```bash
git clone https://github.com/edimehtobechukwu/wegway.git
cd "Weg Way"
npm install
```

### Running the Development Server

Start the local development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173/` (or the port specified in your terminal) to see the application in action.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **Forms**: [React Hook Form](https://react-hook-form.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Carousel**: [Embla Carousel](https://www.embla-carousel.com/)

## 🎨 Design Philosophy

Weg Way embraces a premium design aesthetic, prioritizing vibrant interfaces, modern typography, dynamic micro-animations, and uncompromised accessibility. Every aspect of the UI is crafted to create a lasting impression and a seamless user experience.
